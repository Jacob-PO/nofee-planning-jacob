## 2) 물리학 제1법칙(관성의 법칙) 관점
- 고객의 기본 상태는 "구매 보류"라는 관성이다.
- 관성을 깨려면 외부 힘이 필요하지만, 큰 한 방보다 **작고 지속적인 힘의 합**이 더 안정적으로 속도를 바꾼다.
- 화해의 상세 화면은 **미시적 힘(원자)**을 연속으로 배치해 관성을 구매 방향으로 전환한다.

핵심 은유:
- 관성 = "지금은 안 사도 된다"라는 심리
- 힘 = 숫자, 증거, 개인화, 혜택, 안전장치, 즉시 행동 버튼
- 마찰 = 불확실성(품질/적합성/가격)

## 3) 구매 전환전략의 본질
구매 전환의 본질은 **불안을 줄이고, 확신을 수치화해, 바로 행동하게 하는 것**이다.

- 품질 불안 → 리뷰 수치와 AI 요약으로 객관화
- 적합성 불안 → 피부 타입/성분 매칭으로 자기 검증
- 가격 불안 → 할인율, 포인트, 1+1, 타임딜로 가치 증명
- 행동 마찰 → 상시 노출된 구매 버튼으로 최소 경로 제공

## 5) 원자 결합(분자) = 전환 모듈
원자들은 모듈 단위로 결합되어 순차적으로 힘을 전달한다.

- 제품 핵심 모듈: 이미지 슬라이드 + 평점 + 가격 + CTA
- 가격/혜택 모듈: 할인율 + 최대 혜택가 + 포인트 + 1+1
- 신뢰 모듈: 평점 분포 + 리뷰 수 + AI 리뷰 요약 + 리뷰 토픽
- 적합성 모듈: 성분 위험도 + 전성분 + 피부 타입/목적별 성분
- 안전/배송 모듈: 정품 인증 + 무료배송 + 배송 안내
- 전환 유지 모듈: 연관제품 추천 + 리뷰 작성 유도

## 6) 힘의 시퀀스(전환 흐름)
1. 홈에서 타임딜/only화해로 관심 속도 부여
2. 상세에서 이미지/평점/할인으로 1차 관성 파괴
3. 가격 혜택/포인트로 가치 확신 강화
4. AI 리뷰 요약/토픽으로 품질 불안 해소
5. 성분/피부타입 매칭으로 적합성 검증
6. 리뷰 상세와 실사진으로 최종 신뢰 증폭
7. 고정 CTA로 즉시 구매 행동 유도
8. 리뷰 작성 루프로 신뢰 에너지 재생산

## 7) 결론 - 원자 단위 본질
화해의 구매 전환전략은 **"불확실성(마찰) 감소"를 수치와 개인화로 반복 압축하는 것**이다.
관성에 맞서는 방식은 대형 메시지가 아니라 **원자 수준의 미세한 힘을 연속적으로 누적**시키는 설계다.
그 결과, 고객은 스스로 결론을 내렸다고 느끼면서도 구매 방향으로 속도가 바뀐다.
