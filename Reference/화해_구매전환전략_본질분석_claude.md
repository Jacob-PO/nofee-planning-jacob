# 화해 구매 전환 전략 - 물리학 제1법칙 기반 본질 분석

## 분석 기준
- 분석 대상: 화해_상세 폴더 16개 스크린샷
- 분석 프레임워크: 뉴턴 제1법칙 (관성의 법칙)
- 목적: 전환율 향상을 위한 원자 단위 전략 추출

---

## 물리학 제1법칙 적용

> **관성의 법칙**: 정지한 물체는 외부 힘이 작용하지 않으면 정지 상태를 유지한다.

```
고객 상태: "구매하지 않음" (기본 상태)
전환 조건: F_net = F_motivation - F_friction > 0
```

화해는 **마찰을 최소화**하면서 **동기를 극대화**하는 구조를 설계했다.

---

## 전환 공식의 원자 분해

```
F_motivation = 가치 인식 + 신뢰 + 긴급성 + 개인화
F_friction = 불확실성 + 리스크 공포 + 선택 비용 + 인지 부하
```

화해의 각 UI 요소가 이 공식의 어떤 변수에 작용하는지 분석한다.

# Part 1: 동기 증폭 (F_motivation ↑)

## 원자 #1: 가격 앵커링 시스템

### 관찰 (이미지 07, 08)
```
한정특가 ~~62,000원~~
55% 27,900원
무료배송

27,699원 ← 최대 혜택가
3,890P 최대 적립
```

### 원자 분해

| 요소 | 역할 | 물리학적 해석 |
|------|------|--------------|
| ~~원가~~ 취소선 | 손실 프레이밍 | "안 사면 55% 손해" |
| 55% 큰 글씨 | 가치 인식 극대화 | F_motivation +50 |
| 최대 혜택가 박스 | 최종 가격 명확화 | F_friction -20 (불확실성 제거) |
| P 적립 강조 | 추가 가치 인식 | F_motivation +10 |

### 본질
```
"얼마인지" 보여주는 게 아니라
"얼마나 이득인지" 보여준다
```

**원자 공식**: `[원가 앵커] + [할인율 강조] + [추가 혜택] = 가치 인식 극대화`
