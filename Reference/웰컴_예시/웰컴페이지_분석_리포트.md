## 3. 핵심 패턴 분석

### 3.1 로그인 방식 우선순위

| 순위 | 로그인 방식 | 사용 앱 |
|---|---|---|
| 1위 | **카카오** | 지그재그, 무신사, 화해, 크림 |
| 2위 | **네이버** | 컬리, 화해 |
| 3위 | **Apple** | 모든 앱 (iOS 필수) |
| 4위 | **휴대폰 인증** | 쿠팡 |
| 5위 | **이메일** | 크림, 무신사 |

### 3.2 웰컴 혜택 유형

| 혜택 유형 | 사용 앱 | 효과 |
|---|---|---|
| **% 할인 쿠폰** | 지그재그 (30%) | 가장 직관적 |
| **금액 쿠폰** | 크림 (10,000원, 5,000원) | 구체적 |
| **드로우/응모** | 크림 (100원 상품) | 게임화 |
| **생일 쿠폰** | 크림 | 개인화 |
| **무료배송** | 지그재그 | 추가 혜택 |

### 3.3 CTA 버튼 색상 표준

| 플랫폼 | 색상 | HEX 코드 |
|---|---|---|
| 카카오 | 노란색 | #FEE500 |
| 네이버 | 초록색 | #03C75A |
| Apple | 검정색 | #000000 |
| 서비스 메인 | 브랜드 컬러 | - |

## 4. 노피 웰컴 페이지 개선 제안

### 4.1 즉각 적용 가능한 개선점

#### A. 혜택 강조 (지그재그 참고)
```
현재: 로그인 버튼만 존재
개선: "견적 신청하면 최대 OO만원 할인!" + 비주얼
```

#### B. 시간 강조 (지그재그 참고)
```
현재: "카카오로 로그인"
개선: "카카오로 3초 만에 시작하기"
```

#### C. 비회원 옵션 (무신사 참고) ⭐ 핵심
```
현재: 로그인 필수
개선: "비회원으로 견적 신청" 옵션 추가
```
> **근거**: 퍼널 분석에서 Welcome → Login 53.7% 이탈 발생

#### D. 최근 로그인 표시 (화해 참고)
```
현재: 없음
개선: "카카오로 마지막 로그인 했어요" 표시
```

### 4.2 중장기 개선 제안
