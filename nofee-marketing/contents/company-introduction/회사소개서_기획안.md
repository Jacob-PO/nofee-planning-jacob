## SECTION 2: 시장의 구조적 문제

**대한민국 휴대폰 개통 시장, 무엇이 문제인가**

### 정보 비대칭

고객의 99.9%는 휴대폰 가격표를 본 적이 없습니다.
판매점만 알고, 고객은 모릅니다.
정보 접근 자체가 불가능합니다.

우리 데이터가 보여준 현실:
고객들은 평균 3.1분 동안 서비스를 탐색합니다.
5.5개의 페이지를 봅니다.
이탈률은 2.87%에 불과합니다.

이것이 의미하는 것:
고객들은 정보를 찾고 있었습니다.
하지만 어디에도 없었습니다.

### 불투명한 가격

판매점마다 다른 가격
통신사별, 요금제별, 기종별, 색상별 모두 다른 마진
전화 상담으로만 확인 가능
비교 불가능

결과: "휴대폰 개통은 비싸다"는 고정관념

### 무너진 신뢰

우리가 증명한 신뢰 회복:

참여율 97% - 고객들이 콘텐츠를 끝까지 봅니다
실제 개통 전환율 7.49% - 고객들이 실제로 개통합니다
실제 리뷰 작성 - 고객들이 다시 찾아옵니다

### 접근성 장벽

기존 방식:
판매점마다 전화 문의
대기 시간
영업시간 제약
지역 제약

우리의 해결:
모바일 84.1% - 언제 어디서나
월평균 만 건 이상의 세션 - 24시간 접근
모든 가격표 공개 - 실시간 비교

## SECTION 6: 3번의 시도, 30배의 성장

**버전 1 - 일반견적 (2025년 7월 ~ 8월)**

개통율: 2.8%

데이터:
- 고객이 원하는 기종을 입력
- 전체 판매점의 평균 견적 제공
- 고객이 직접 판매점 선택

문제 발견:
고객은 "평균 가격"이 아니라 "최저가"를 원했습니다.

**버전 2 - 캠페인 견적 (2025년 7월 ~ 8월)**

개통율: 0.5%

데이터:
- 인기 기종 캠페인 형식 제공
- 최저가 자동 표시
- 검색 기능 없음

문제 발견:
고객은 자신이 원하는 기종을 "검색"하고 싶어했습니다.
선택지를 제한하면 이탈합니다.

**버전 3 - 동네성지 (2025년 9월 ~ 10월 베타테스트)**

평균 개통율: 6.0%
최고 기록: 15.3% (10월)

개선 사항:
- 고객이 원하는 기종 검색 가능
- 지역별 최저가 자동 표시
- 내 동네 판매점 우선 노출
- 실시간 가격표 업데이트

결과:
0.5%에서 15.3%까지 - 30배 향상

**그래프로 보는 개선**

V1 (7-8월): 2.8%
V2 (7-8월): 0.5%
V3 (9월): 2.2%
