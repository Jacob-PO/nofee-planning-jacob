## 1. 분석 프레임워크 요약

### 1.1 핵심 물리학 공식

```
전환 발생 조건:
F_net = F_motivation - F_friction > 0

동기력 (F_motivation):
= (신뢰 × 가치 × 긴급성) / 인지부하

마찰력 (F_friction):
= 불확실성 + 리스크공포 + 선택비용 + 인지부하

엔트로피:
S = log(n)  // 선택지 n이 많을수록 S↑ → 이탈↑
```

### 1.2 쿠팡의 핵심 전략: "로켓" 물리학

쿠팡은 **로켓배송**이라는 이름처럼 물리학적 원리를 직관적으로 구현:

| 물리학 원리 | 쿠팡 구현 |
|------------|----------|
| 가속도 극대화 | "내일(월) 도착", "지금 구매 시" |
| 마찰력 제거 | "무료배송", "무료반품", "저렴한 반품" |
| 불확정성 해소 | "100명 이상 구매", "스마트 상품 요약 AI" |
| 에너지 보존 | 상품카드 → 상세 정보 일관성 |
| 충격량 집중 | Hero 영역에 할인율 + 배지 + 가격 집중 |

## 3. 원자별 심층 분석

### 3.1 가치 인식 원자 (A01~A08)

#### A01: 할인율 강조 - 앵커링의 가속도

**쿠팡 구현** (쿠팡_02, 쿠팡_04):
```
지금 42% 할인 중
30% 30% 할인
48% 48% 할인
55% 55% OFF
```

**물리학 분석**:
- **뉴턴 제2법칙**: a(가속도) = 가치 인식 변화율
- 빨간색 + 굵은 글씨 + 취소선 = 시각적 가속도 극대화
- "지금" 키워드 = 즉시성 가속도 부가
- **앵커링 효과**: 원가 → 할인가 낙차가 클수록 가속도↑

**힘 기여**: F = +0.9

```
계산:
시각적 강조도 = 0.9 (빨간색 + 굵은 글씨)
할인율 평균 = 40%
a = 0.9 × 0.4 × 2.5 = 0.9
```

---

#### A03: 와우카드 혜택 - 다층 보상 에너지

**쿠팡 구현** (쿠팡_01, 쿠팡_02):
```
쿠팡 와우카드          더보기 >
1.2%  외식, 주유 등 최대     4%  쿠팡에서 최대
      1.2% 캐시적립               4% 캐시적립
```

**물리학 분석**:
- **에너지 중첩**: 여러 혜택이 겹쳐 총 에너지 증가
- 1.2% + 4% = 5.2% 추가 가치 인식
- **운동에너지 전환**: 잠재 혜택 → 실제 구매 동기
- 황금색 카드 이미지 = 프리미엄 인식 = 추가 에너지

**힘 기여**: F = +0.5

```
계산:
기본 캐시백 = 4%
추가 혜택 = 1.2%
프리미엄 인식 = 0.1
F_A03 = (0.04 + 0.012) × 10 + 0.1 = 0.62 ≈ +0.5
```

---

#### A05: 내일 배지 - 즉시성의 가속도 극대화

**쿠팡 구현** (쿠팡_01, 쿠팡_04):
```
로켓배송 내일(월) 12/22 도착
무료배송
```

**물리학 분석**:
- **시간 = 거리/속도**: 배송 시간↓ = 속도↑ = 가속도↑
- "내일" = 24시간 내 = 최대 즉시성
- **심리적 시간 압축**: 미래의 보상이 현재화됨
- 구체적 날짜(12/22) = 불확정성↓

**힘 기여**: F = +0.9

```
계산:
시간 근접성 = 1/1 = 1.0 (내일 = 1일)
날짜 명확성 = 0.9 (구체적 날짜)
무료배송 보너스 = 0.0 (마찰 감소로 별도 계산)
F_A05 = 1.0 × 0.9 = 0.9
```

#### A06: 특가전용 배지 - 긴급성의 충격량

**쿠팡 구현** (쿠팡 PC 메인):
```
[특가진행중] (빨간 배지)
[대통령상수상]
[지금 구매시 사은품 14000원 상당]
```

**물리학 분석**:
- **충격량 J = F × Δt**: 강한 시각 자극 × 짧은 인식 시간
- 빨간색 배지 = 경고/긴급 신호 = 즉각 주목
- **색상 대비**: 흰 배경에 빨간 배지 = 시각적 돌출
- 복수 배지 = 다중 충격량 중첩

**힘 기여**: F = +0.8

```
계산:
색상 충격도 = 0.9 (빨간색)
배지 수 = 3
주목 시간 = 0.3초
J_A06 = 0.9 × 3 × (1/0.3) × 0.1 = 0.9 ≈ +0.8
```

#### B03: 구매자 수 - 사회적 증거의 질량 극대화

**쿠팡 구현** (쿠팡_01, 쿠팡_04):
```
한 달간 100명 이상 구매했어요
한 달간 1만명 이상 구매했어요
```

**물리학 분석**:
- **뉴턴 제2법칙**: m(질량) = 사회적 증거의 양
- 1만명 = 거대한 질량 = 강력한 전환력
- **노란색 배경**: 하이라이트 효과 = 시각적 강조
- "한 달간" = 시간 한정 = 신선도 보장

**힘 기여**: F = +0.9

```
계산:
m = log(10,000) = 4.0 (정규화된 질량)
시간 한정 신선도 = 0.9
시각적 강조 = 0.2
F_B03 = 4.0 × 0.9 × 0.3 + 0.2 = 1.28 → cap at 0.9
```

### 3.2 신뢰 구축 원자 (B01~B06)

#### B04: Apple 공식 - 권위의 전자기력

**쿠팡 구현** (쿠팡_04):
```
Apple >
브랜드 직매입 ◉
★★★★★ (3,172)

Apple 아이폰 17 Pro 자급제
```

**물리학 분석**:
- **권위의 쿨롱 인력**: 강력한 브랜드 = 강한 인력
- Apple 로고 + "브랜드 직매입" = 정품 보장 이중 인력
- **전자기장 확산**: Apple 권위가 쿠팡으로 전이
- 클릭 가능한 브랜드 링크 = 탐색 경로 제공

**힘 기여**: F = +0.8

```
계산:
브랜드 권위 q1 = 1.0 (Apple = 최고)
플랫폼 신뢰 q2 = 0.8 (쿠팡)
심리적 거리 r = 1.0 (직매입)
F_B04 = k × (1.0 × 0.8) / 1.0² = 0.8k ≈ +0.8
```

확정 후 = 0.3 (공식 인증)
불확정성 감소 = 0.7
F_B05 = 0.7 (마찰력 감소로 변환)
```

### 3.3 마찰력 감소 원자 (E01~G02)

#### E01: 스마트 상품 요약 - AI의 엔트로피 감소

**쿠팡 구현** (쿠팡_03):
```
✦ 스마트 상품 요약               by Coupang AI ⓘ

아이폰 15용 0.3mm 초슬림 맥세이프 케이스. 변색 없는 반투
명 디자인과 강력한 자력, 카메라 보호 기능을 갖췄습니다.

사양
• 브랜드: 가꼬싶다
• 모델: 변색없는 0.3mm 엣지슬림 맥세이프 케이스
• 호환성: 아이폰 15
• 유형: 맥세이프 슬림 케이스

[👍] [👎]    전체 요약 보기 ∨
```

**물리학 분석**:
- **열역학 제2법칙 역전**: 리뷰/스펙 무질서 → 핵심 요약으로 질서화
- AI = 엔트로피 감소 엔진
- 불릿 포인트 = 정보 구조화 = 인지부하↓
- 피드백 버튼(👍👎) = 신뢰도 검증 장치

**힘 기여**: F = +0.7 (마찰력 -0.7)

```
계산:
원본 정보량 = 상품 설명 + 리뷰 수백개
압축 후 = 4개 불릿 포인트
엔트로피 감소 = log(N) - log(4) ≈ 2.0
마찰력 감소 = 2.0 × 0.35 = 0.7
```

#### E02: 적용금액 계산 - 최종가의 불확정성 해소

**쿠팡 구현** (쿠팡_01):
```
9,800원 ~~14,000원~~ ⓘ
10월 할인 적용 9,810원
```

**물리학 분석**:
- **양자 불확정성 해소**: "결국 얼마야?" → 정확한 숫자
- 원가 취소선 + 할인가 = 가치 앵커링
- "10월 할인 적용" = 시간 한정 + 추가 혜택 명시
- 정보 아이콘(ⓘ) = 투명성 신호

**힘 기여**: F = +0.6

```
계산:
가격 불확정성 원래 값 = 0.8 (할인, 배송비, 적립 등 복잡)
계산 후 = 0.2 (최종가 명시)
불확정성 감소 = 0.6
F_E02 = 0.6
