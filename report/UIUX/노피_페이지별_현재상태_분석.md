## 1. 분석 프레임워크 개요

### 1.1 물리학 기반 전환 공식

```
F_net = F_motivation - F_friction

F_motivation = 가격절감 + 신뢰신호 + 긴급성 + 사회적증거
F_friction = 인지부하 + 불확실성 + 선택마비 + 이탈저항
```

### 1.2 적용 물리 법칙

| 법칙 | UX 적용 |
|------|---------|
| **뉴턴 제1법칙 (관성)** | 사용자의 기존 습관 극복 |
| **뉴턴 제2법칙 (F=ma)** | 동기부여 힘이 클수록 전환 가속 |
| **운동량 보존** | 페이지 간 신뢰 정보 계승 |
| **열역학 제2법칙 (엔트로피)** | 선택지 최소화로 혼란 감소 |
| **양자 터널링** | 낮은 장벽으로 전환 확률 증가 |

Product  ████████████████████████████░░  +2.9  (최고)

### 6.2 개선 우선순위

#### 🔴 높음 (High Priority)
| 페이지 | 개선 항목 | 예상 효과 |
|--------|----------|----------|
| Deal | Hero 영역 즉시 CTA 추가 | F_net +0.5 |
| Deal | 안심 문구 톤 강화 ("100% 환불 보장") | F_net +0.3 |
| Compare | 비교 상품 수 3개 제한 | 엔트로피 -0.4 |

#### 🟡 중간 (Medium Priority)
| 페이지 | 개선 항목 | 예상 효과 |
|--------|----------|----------|
| Home-v2 | 긴급성 요소 강화 (재고, 마감) | F_net +0.4 |
| Product | Deal 연결 CTA 강화 | 운동량 +0.3 |
| Compare | 점수 기준 설명 추가 | 불확실성 -0.3 |

#### 🟢 낮음 (Low Priority)
| 페이지 | 개선 항목 | 예상 효과 |
|--------|----------|----------|
| 전체 | 일관된 신뢰 배지 디자인 | 브랜드 신뢰 +0.2 |
| 전체 | 로딩 상태 최적화 | 이탈률 -0.1 |

### 6.3 운동량 보존 체크

#### 페이지 전환 시 정보 계승 현황

```
Home-v2 → Deal
├── 할인율: ✅ 계승됨
├── 판매점 평점: ✅ 계승됨
├── 구매자 수: ✅ 계승됨
└── 리뷰 수: ✅ 계승됨

Product → Deal
├── 상품 평점: ✅ 계승됨
├── AI 키워드: ⚠️ 부분 계승
└── 스펙 정보: ⚠️ 부분 계승

Compare → Deal
├── 비교 점수: ❌ 미계승
└── 선택 이유: ❌ 미계승
```

### 6.4 결론

Deal 페이지가 이전 분석 대비 **F_net +3.1** 크게 개선되어 전환 최적화 수준에 도달했습니다.
특히 신뢰 신호(별점, 리뷰, 구매자 수)가 모두 Visible하게 표시되고,
단일 CTA로 엔트로피가 최소화된 점이 주요 개선 사항입니다.

향후 Hero 영역 즉시 CTA 추가와 Compare 페이지 엔트로피 감소에 집중하면
전체 전환율을 추가로 향상시킬 수 있습니다.
