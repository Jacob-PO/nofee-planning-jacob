## 1. 물리학 분석 프레임워크

### 1.1 적용 법칙 전체 목록

| 분야 | 법칙 | 전환 맥락 적용 |
|------|------|---------------|
| **고전역학** | 뉴턴 제1법칙 (관성) | 미전환 상태 유지 성향 |
| | 뉴턴 제2법칙 (F=ma) | 전환력 = 동기 × 가속도 |
| | 뉴턴 제3법칙 (작용-반작용) | 마찰을 줄이면 반발도 줄어듦 |
| | 운동량 보존 (p=mv) | 사용자 관심의 누적/소멸 |
| | 충격량 (J=FΔt) | 강한 힘 × 짧은 시간 = 순간 전환 |
| **에너지** | 운동에너지 (½mv²) | 스크롤 속도 기반 관심도 |
| | 위치에너지 (mgh) | 퍼널 위치별 전환 잠재력 |
| | 에너지 보존 | 신뢰 에너지의 이동/손실 |
| **열역학** | 제1법칙 (에너지 보존) | 정보 에너지의 총량 |
| | 제2법칙 (엔트로피 증가) | 복잡성 → 이탈 |
| | 제3법칙 (절대영도) | 전환 의지 0 상태 |
| **전자기학** | 쿨롱 법칙 | 신뢰-불신의 인력/척력 |
| | 자기 유도 | 사회적 증거의 전파 |
| **양자역학** | 불확정성 원리 | 가격/조건 불확실성 |
| | 파동-입자 이중성 | 탐색자 vs 구매자 이중 상태 |
| | 양자 터널링 | 마찰 장벽 돌파 |
| **유체역학** | 베르누이 원리 | 퍼널 병목 → 속도 증가 |
| | 연속 방정식 | 유입 = 유출 + 이탈 |

### 1.2 핵심 공식

```
전환 발생 조건:
F_net = F_motivation - F_friction > 0

동기력:
F_motivation = (신뢰 × 가치 × 긴급성) / 인지부하

마찰력:
F_friction = 불확실성 + 리스크공포 + 선택비용 + 인지부하

에너지 보존:
E_total = E_kinetic (관심) + E_potential (잠재전환) + E_lost (이탈)

엔트로피:
S = -Σ(p_i × log(p_i))  // 선택지가 많을수록 S↑ → 이탈↑
```

### 3.2 원자별 문제 정의

#### 원자 #1: Welcome 페이지 진입

**현재 코드**: `app/(auth)/welcome/page.tsx`
```tsx
<button onClick={handleKakaoLogin}>
  <KakaoIcon />
  카카오로 5초만에 시작
</button>
<p className="mt-3 text-center text-xs text-gray-500">
  전화 연락 없이 채팅으로 간편하게 상담받으세요
</p>
```

**물리학 분석**:

| 법칙 | 현재 상태 | 문제 |
|------|----------|------|
| 뉴턴 제1법칙 (관성) | 버튼 1개, 메시지 1개 | 관성 파괴력 약함 |
| F=ma (가속도) | 가속도 a 낮음 | 전환력 F 부족 |
| 에너지 보존 | 위치에너지만 존재 | 운동에너지 부여 필요 |
| 엔트로피 | S ≈ 0 (선택지 1개) | **양호** |

**원자 문제**:
- 🔴 **동기 부족**: "왜 지금 가입해야 하는가?"에 대한 답 없음
- 🔴 **신뢰 부족**: 사회적 증거(N명 사용 중) 없음
- 🔴 **긴급성 부족**: 시간 제한/혜택 없음
- 🟢 **마찰 낮음**: 버튼 1개 (엔트로피 최소)

**힘 계산**:
```
F_motivation = 0.3 (가치: 전화연락 없음) + 0.1 (신뢰: 없음) + 0 (긴급성)
             = 0.4

F_friction = 0.2 (인지부하: 낮음) + 0.1 (리스크: 카카오 신뢰)
           = 0.3

F_net = 0.4 - 0.3 = +0.1 → 약한 전환력
```

#### 원자 #2: 약관 동의 (signup/terms)

**현재 코드**: `app/(auth)/signup/terms/page.tsx`
```tsx
<Title>서비스 이용 약관을 확인해주세요.</Title>
<TermsList>
  <Label>전체 동의</Label>
  {TERMS.map((term) => (
    <Label>{term.label}</Label>  // 5개 체크박스
  ))}
</TermsList>
<Button $disabled={!requiredChecked}>다음으로</Button>
```

**물리학 분석**:

| 법칙 | 현재 상태 | 문제 |
|------|----------|------|
| 뉴턴 제1법칙 | 체크 5개 = 마찰 5단위 | 관성 유지됨 |
| 엔트로피 | S = log(2^5) = 5 | **높음 = 이탈↑** |
| 열역학 2법칙 | 복잡성 증가 → 무질서 증가 | 포기 유발 |

**원자 문제**:
- 🔴 **마찰 과다**: 5개 체크박스 = 5번의 의사결정
- 🔴 **가치 불명확**: "왜 이 약관에 동의해야 하는가?"
- 🔴 **진행 표시 없음**: 전체 단계 중 어디인지 모름

**힘 계산**:
```
F_motivation = 0.2 (전 단계 운동량 유지)
F_friction = 0.5 (체크박스 5개) + 0.2 (약관 읽기 부담)
           = 0.7

F_net = 0.2 - 0.7 = -0.5 → 이탈 발생!
```

- 🟡 **마찰 중간**: 2단계지만 드롭다운 선택으로 간소화됨

**힘 계산**:
```
F_motivation = 0.3 (거의 완료 = 손실 회피)
F_friction = 0.3 (선택 2단계) + 0.2 (왜 필요한지 모름)
           = 0.5

F_net = 0.3 - 0.5 = -0.2 → 이탈 발생!
```

```
             = 2.3

F_friction = 0.3 (클릭 비용)
           = 0.3

F_net = 2.3 - 0.3 = +2.0 → 강한 전환력 ✓
```
