## 1. 분석 프레임워크 요약

### 1.1 핵심 물리학 공식

```
전환 발생 조건:
F_net = F_motivation - F_friction > 0

동기력 (F_motivation):
= (신뢰 × 가치 × 긴급성) / 인지부하

마찰력 (F_friction):
= 불확실성 + 리스크공포 + 선택비용 + 인지부하

엔트로피:
S = log(n)  // 선택지 n이 많을수록 S↑ → 이탈↑
```

### 1.2 적용 물리학 법칙

| 법칙 | 전환 맥락 |
|------|----------|
| 뉴턴 제1법칙 (관성) | 미구매 상태 유지 → 강한 초기 힘 필요 |
| 뉴턴 제2법칙 (F=ma) | 동기(m) × 가속도(a) = 전환력(F) |
| 운동량 보존 (p=mv) | 페이지 간 신뢰/관심 유지 |
| 충격량 (J=FΔt) | 강한 힘을 짧은 시간에 집중 |
| 에너지 보존 | 축적된 신뢰 에너지 손실 방지 |
| 열역학 제2법칙 | 엔트로피 증가 → 이탈 증가 |
| 양자 불확정성 | 가격/조건 불확실성 해소 |
| 양자 터널링 | 마찰 장벽 낮추면 돌파 확률↑ |
| 쿨롱 법칙 | 신뢰-불신의 인력/척력 |

## 3. 원자별 심층 분석

### 3.1 가치 인식 원자 (A01~A07)

#### A01: 숫자 평점 - 사회적 증거의 질량화

**화해 구현** (화해_01, 화해_07):
```
★ 4.73 (22,287)
★ 4.64 (305)
```

**물리학 분석**:
- **뉴턴 제2법칙**: F = ma에서 m(질량) = 사회적 증거
- 22,287명의 평가 = 거대한 질량
- 질량이 클수록 같은 가속도에서 더 큰 힘 발생
- **운동량 보존**: p = mv → 큰 질량은 쉽게 멈추지 않음

**힘 기여**: F = +0.8

```
계산:
m = log(22,287) = 4.35 (정규화)
a = 4.73/5 = 0.95 (평점 기반 가속도)
F_A01 = 4.35 × 0.95 × 0.2 = 0.83 ≈ +0.8
```

#### A02: 할인율 앵커링 - 가치 인식의 가속도

**화해 구현** (화해_07, 화해_08):
```
한정특가 62,000원
55% 27,900원
무료배송
```

**물리학 분석**:
- **뉴턴 제2법칙**: a(가속도) = 인식된 가치 변화율
- 55% 할인 = 가속도 0.55 (정상 대비 55% 더 빠른 의사결정)
- **위치에너지 → 운동에너지 전환**: 높은 정가에서 낮은 실제가로 "떨어지는" 감각
- **손실 회피**: 62,000원 → 27,900원 = 34,100원 "얻는" 느낌

**힘 기여**: F = +0.9

```
계산:
가격 낙차 = 62,000 - 27,900 = 34,100원
낙차 비율 = 34,100 / 62,000 = 0.55
F_A02 = 0.55 × 1.5 (앵커링 승수) = 0.83 ≈ +0.9
```

#### A05: 랭킹 배지 - 충격량의 극대화

**화해 구현** (화해_01, 화해_07):
```
1위 AWARDS 배지 (우측 상단)
뷰티어워즈 선정
```

**물리학 분석**:
- **충격량 J = F × Δt**: 강한 힘을 짧은 시간에 전달
- "1위"라는 단어 = 즉각적으로 인식되는 강력한 신호
- **권위의 전자기력**: 권위 있는 정보는 인력(引力)처럼 작용
- 눈에 띄는 위치(우측 상단) = 첫 0.3초 내 인식

**힘 기여**: F = +0.7

```
계산:
F = 1.0 (1위 = 최대 권위)
Δt = 0.3초 (즉각 인식)
J_A05 = 1.0 × (1/0.3) × 0.2 = 0.67 ≈ +0.7
```

#### C01: 타임딜 배너 - 긴급성의 가속도

**화해 구현** (화해_03):
```
오늘만 이 가격!
24H 선착순 초특가
화해페스타 타임딜
```

**물리학 분석**:
- **가속도 a = Δv/Δt**: 시간이 짧을수록 가속도 급증
- 24H = 남은 시간이 짧음 = 높은 가속도
- **열역학적 해석**: 시간 제한 = 엔트로피 감소 (선택 기회 축소)
- 선착순 = 경쟁 상황 = 추가 가속도

**힘 기여**: F = +0.9

```
계산:
시간 압박 = 1/24 = 0.042 (일 단위)
경쟁 압박 = 1.5 (선착순 승수)
긴급성 = 0.042 × 1.5 × 10 = 0.63
색상 대비(핑크 배너) = +0.3
F_C01 = 0.63 + 0.3 = 0.93 ≈ +0.9
```

### 3.2 신뢰 구축 원자 (B01~B09)

#### B03: AI 리뷰 분석 - 엔트로피 감소

**화해 구현** (화해_09):
```
AI가 분석한 리뷰

좋았던 점          아쉬운 점
보습이 잘 돼요 137   유분이 있어요 18
수분이 있어요 110   알러지 반응이 와요 7
흡수가 잘 돼요 101  담담해요 6
```

**물리학 분석**:
- **열역학 제2법칙 역전**: 일반적으로 엔트로피(무질서)는 증가
- AI 분석 = 22,287개 리뷰 → 6개 핵심 포인트로 압축
- **엔트로피 감소**: S = log(22,287) → log(6) = 4.35 → 0.78
- 무질서 → 질서 = 의사결정 비용 급감

**힘 기여**: F = +0.6

```
계산:
원본 엔트로피 = log(22,287) = 4.35
압축 후 엔트로피 = log(6) = 0.78
엔트로피 감소 = 4.35 - 0.78 = 3.57
마찰력 감소 = 3.57 × 0.15 = 0.54 ≈ +0.6
```

### 3.3 마찰력 감소 원자 (E01~G02)

#### E01: 성분 위험도 - 불확정성의 수치화

**화해 구현** (화해_11):
```
성분 구성
● 1-2 낮은 위험  ● 3-6 중간 위험  ● 7-10 높은 위험  ● 등급 미정

[시각적 막대 그래프]
전체 성분: 32개
20가지+ 주의성분
알레르기 주의성분
내 관심 성분
내 피할 성분
```

**물리학 분석**:
- **양자 불확정성 → 고전적 확정**: "안전한가?" → 1-10 스케일로 확정
- 숫자 스케일 = 파동함수 붕괴 = 불안 해소
- **색상 코딩**: 파랑(안전) - 노랑(주의) - 빨강(위험) = 직관적 이해
