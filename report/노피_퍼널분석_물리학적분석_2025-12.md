##### 🥇 1위: 홈 → Product (51.6% 이탈, 19,836명 손실)

```
문제: 홈에 들어왔는데 상품 페이지로 안 감
- 38,439명 중 19,836명이 상품을 보지 않고 이탈
- 연간 약 2만 명의 잠재고객 손실

원인 추정:
□ CTA 불명확 ("뭘 클릭해야 하지?")
□ 가치 명제 약함 ("이게 뭔데?")
□ 첫인상에서 신뢰 부족
□ 모바일 UX 문제 (84.7% 모바일)
□ 로딩 속도 또는 초기 화면 문제

물리학 해석:
- F_motivation ≈ 0 (동기력 부재)
- 초기 운동에너지 부족으로 관성 극복 실패
```

##### 🥈 2위: Welcome → 로그인 (53.7% 이탈, 9,931명 손실)

```
문제: 로그인 페이지까지 왔는데 로그인 안 함
- 18,502명 중 9,931명이 로그인 버튼 안 누름
- 연간 약 1만 명의 고의도 사용자 손실

원인 추정:
□ "왜 로그인해야 하지?" (가치 명제 없음)
□ "개인정보 수집하려는 건가?" (불신)
□ 카카오 로그인 팝업에서 취소
□ 로그인 후 뭐가 되는지 불명확
□ 비회원 옵션 없음

물리학 해석:
- E_barrier >> E_user (에너지 장벽)
- 양자 터널링 불가 → 비회원 경로 필요
```

## 2. 물리학적 진단 프레임워크

### 2.1 전환 발생 조건

```
전환 발생 조건:
F_net = F_motivation - F_friction > 0

동기력 (F_motivation):
= (신뢰 × 가치 × 긴급성) / 인지부하

마찰력 (F_friction):
= 불확실성 + 리스크공포 + 선택비용 + 인지부하

엔트로피:
S = log(n)  // 선택지가 많을수록 S↑ → 이탈↑
```

### 2.2 적용 법칙

| 물리학 법칙 | 적용 해석 |
|------------|----------|
| **뉴턴 제1법칙 (관성)** | 사용자는 현재 상태 유지하려 함 → 강한 힘 필요 |
| **뉴턴 제2법칙 (가속)** | F = ma → 동기력이 클수록 전환 가속 |
| **열역학 제2법칙** | 엔트로피 증가 → 선택지 많으면 이탈 증가 |
| **위치에너지** | 목표와의 거리 → 불확실성 = 위치에너지 |
| **양자 터널링** | 장벽을 뚫는 경로 → CTA 단순화 |
| **전자기 인력** | 신뢰 = 인력 → 거리 가까울수록 강함 |

해석: F_net < 0이지만 "호기심"이라는 추가 동력으로 돌파
