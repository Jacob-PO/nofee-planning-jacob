개요

이번 개선은 “홈에서 딜로 넘어가는 비율이 낮다”와 “Product에서 딜로 이동이 적다”는 데이터를 근거로 진행했습니다. GA4 데이터를 2025-12-08~12-21로 다시 뽑아 원인과 방향을 정리했습니다.

핵심 변경사항
• Deal: 가격/혜택을 첫 화면 맨 위에 배치, 상단 CTA 추가, 비슷한 상품 상단 이동, 섹션/CTA/스크롤 트래킹 추가
• Product: 최저가 딜 섹션 상단 이동, 상단/스티키 CTA 문구 통일, 딜 노출 수 확대, CTA/딜 섹션 트래킹 추가
• 공통: 홈→딜 이동 문제를 채널/기기별로 나눠 확인

문제 정의 (데이터 기반)

홈에 들어온 사람은 많지만, 딜로 넘어가는 비율이 낮습니다. 특히 모바일·Paid Social에서 이탈이 큽니다.

  • 홈 방문 1,762세션 중 딜 카드 클릭 194세션(11.0%)
  • 모바일 8.7%, 데스크탑 24.3%
  • Paid Social 5.4%로 가장 낮음
  • 최근 클릭률 급락: 12/16 24.0% → 12/21 1.2%

무엇을 왜/어떻게 바꿨나

Deal 페이지

  • 가격/혜택을 첫 화면 맨 위로 이동: 모바일·Paid Social에서 이탈이 커서 첫 화면에서 바로 가치 전달
  • 비슷한 상품 섹션 상단 이동: 스크롤을 많이 내리기 전에 다른 딜을 비교할 수 있게 함
  • 상단 CTA가 보일 때 하단 CTA 숨김: 두 CTA 동시 노출로 생기는 혼란 방지
  • 섹션/CTA/스크롤/가격 상세 트래킹 추가: 개선 효과를 숫자로 확인 가능

Product 페이지

  • 최저가 딜 섹션 상단 이동: Product에서 딜로 넘어가는 비율이 낮아 상단에 먼저 노출
  • 상단/스티키 CTA 문구 통일: “최저가 딜 보기”로 하나의 행동으로 유도
  • 딜 노출 수 확대(5→10개): 선택지를 늘려 클릭 기회 확대
  • CTA/딜 섹션 노출·클릭 트래킹 추가: 개선 효과를 숫자로 확인

기대되는 데이터 변화

이번 변경은 클릭이 많이 떨어지는 구간에서 가치와 행동을 더 빨리 보여주는 방식입니다.

  • 홈→딜 클릭률 11.0%: 첫 화면 가치 전달 강화로 클릭률 상승 기대
  • Product 첫 방문→딜 전환 6.6%: 딜 섹션 상단/CTA 강화로 이동 증가 기대
  • Deal 상세→견적 클릭 20.9%: 가격/혜택 첫 화면 + 비슷한 상품 상단 배치로 클릭 증가 기대

개선 효과는 새 GA4 이벤트(CTA 노출/클릭, deal_section_view, product_deal_section_view)로 숫자로 확인할 예정입니다.
