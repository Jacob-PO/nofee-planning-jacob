관리 팁

  • 개통정보 입력: 가입유형과 구매유형, 정산 예정일만 입력하면 됩니다. 나머지는 필요시 입력하세요.
  • 정산 상태: 고객의 정산 완료 여부를 체크하고, 필요한 메모를 남기세요.
  • 알림톡: 정산 예정일을 입력하면 자동으로 그 전날 오전 9시에 알림톡이 발송됩니다.
  • 모든 변경사항을 입력한 후 저장 버튼을 클릭하세요.

개발 추진 단계 (v2.1)

Phase 1

메인 탭 네비게이션

4개 탭 구조 (개통정보 입력, 정산, 알림톡, 분석)

상태: 필수

Phase 2

고객관리 탭 페이지

고객 리스트 → 수정 버튼 → 새 탭 (개통정보, 정산, 알림톡)

상태: 필수

Phase 3

미완 작업 경고 & 폼 최적화

상단 경고 카드, 필수 필드만 앞, 자동 계산, 조건부 필드

상태: 필수

Phase 4

표/필터 및 상호작용

고객 목록 4-5컬럼, 의미색 배지, 필터, 탭 전환

상태: 필수

추가 사항

  • 반응형: 모바일/태블릿 대응 (탭 overflow-x, 카드 스택, 전체폭 버튼)
  • 접근성: 필드 라벨, 색상 대비 WCAG 준수, 필드별 검증 메시지
  • 성능: 풀 페이지 로드 최적화, 자동 계산 성능 (debounce 적용)
  • 데이터: 사용자 별 권한 기반 필터/표시 (대리점 매니저 vs 관리자)
