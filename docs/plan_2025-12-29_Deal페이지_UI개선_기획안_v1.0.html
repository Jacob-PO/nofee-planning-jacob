현재 문제점

1. 버튼이 2개라서 헷갈림

기존에는 "판매점과 채팅하기"와 "바로 구매" 두 개의 버튼이 있었습니다. 사용자 입장에서 둘의 차이가 명확하지 않았습니다.

지표 듀얼 CTA 기간 (12/27-28) 변화
CTA 클릭률 17.8% 기존 25.4% 대비 -7.6%p
신청 완료율 39.1% 기존 66.2% 대비 -27.1%p
클릭/사용자 3.91회 기존 2.14회 대비 +1.77회 (혼란 지표)

클릭당 사용자 수가 2.14회에서 3.91회로 늘었습니다. 이건 사용자가 헷갈려서 여러 번 클릭했다는 의미입니다.

2. 신뢰할 만한 정보가 부족했음

Home 페이지에서는 별점, 리뷰 수, 할인율, 구매자 수 같은 정보가 잘 보였는데, Deal 페이지로 넘어오면 이런 정보가 대부분 사라졌습니다.

사용자는 "내가 맞게 들어온 건가?" 하는 의문을 갖게 됩니다.

3. 판매점 정보가 눈에 안 들어옴

판매점 별점이 화면에 보이지 않았습니다. 기술적으로는 존재했지만 SEO용으로 숨겨져 있었습니다. 사용자는 모르는 판매점에 연락하는 것에 불안함을 느꼈습니다.

개선 방향

핵심 전략
1. 버튼을 하나로 통일해서 선택 장애 제거
2. 첫 화면에서 바로 신뢰할 수 있는 정보 보여주기
3. 불안을 줄여주는 안심 문구 추가
CTA 버튼 단일화

변경 전

"판매점과 채팅하기" 버튼
"바로 구매" 버튼

변경 후

"전화 연락 없는 견적받기" 단일 버튼

노피의 핵심 가치인 "전화 없이 편하게"를 버튼 문구에 그대로 담았습니다. 사용자가 고민할 필요 없이 바로 클릭할 수 있습니다.

신뢰 정보 표시

페이지 상단(Hero 영역)에서 바로 볼 수 있도록 다음 정보를 추가했습니다.

  • 상품 별점/리뷰: "4.5점 (1,234개의 상품평)"
  • 판매점 별점: "서울 강남구 > 노피폰 > 4.8"
  • 구매자 수: "최근 1천명 이상 구매했어요"
  • 할인율: 출고가에 취소선을 그어 할인 폭을 시각화
불안 해소 요소

휴대폰 구매는 큰 결정입니다. 사용자의 불안을 줄이기 위해 두 가지를 추가했습니다.

  • 안심 문구: "전화, 카톡 없이 노피 채팅으로만 소통해요"
  • 긴급성: "이 가격은 지금부터 30분 뒤 변동됩니다"
