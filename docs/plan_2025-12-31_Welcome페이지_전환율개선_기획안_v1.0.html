3. 왜 이탈하나? - 현재 문제점

문제 1: Welcome 페이지 - "왜 가입해야 하지?"

기존 Welcome 페이지는 "카카오로 로그인" 버튼만 덜렁 있었습니다.

  • "가입하면 뭐가 좋은데?" → 혜택이 안 보임
  • "내가 보던 상품은?" → Deal 페이지에서 봤던 상품이 사라짐
  • 가입할 이유를 못 느껴서 이탈
문제 2: 약관 동의 - "체크박스가 너무 많아"
  • 체크박스 5개를 하나씩 눌러야 함
  • 전체 몇 단계인지 모름 (진행률 표시 없음)
  • "왜 이걸 동의해야 하지?" 설명이 없음
문제 3: 희망정보 - "이거 꼭 입력해야 해?"
  • Step 1 (지역): 스킵할 수 없음
  • Step 2 (통신사/가격대/제조사): X 버튼으로 스킵 가능
  • "이 정보를 왜 받는지" 설명이 없음
  • 여기까지 오면 이미 지쳐서 이탈

현재 문제점 요약
- 약관 동의 후에야 회원이 생성됨 → 약관에서 이탈하면 회원이 안 됨
- 총 4개 화면(약관 + 지역 + 희망정보 + 완료)을 거쳐야 함
- 진행률 표시 없음 → "아직 얼마나 남았지?" 불안감

요소 현재 개선
로고 없음 상단 중앙 (브랜드 인지)
닫기 버튼 우상단 X 하단 텍스트 링크 ("둘러보기")
카카오 버튼 위치 화면 하단 고정 비주얼 바로 아래 (자연스러운 시선 흐름)
버튼 문구 카카오로 3초 만에 시작하기 동일 유지
에이블리 레이아웃의 장점
1. 로고 → 헤드라인 → 비주얼 → 버튼 순서로 자연스러운 시선 흐름
2. CTA 버튼이 콘텐츠 바로 아래에 있어 클릭 유도 효과 높음
3. 닫기 버튼 대신 "둘러보기" 링크로 이탈 시에도 긍정적 경험 제공

왜 이렇게 바꾸나요?

  • 지금은 약관에서 이탈하면 회원이 아예 안 됨
  • 바꾸면: 일단 회원으로 만들고, 나중에 약관 동의 받기
  • 이탈해도 최소한 회원은 확보됨
백엔드 수정 필요

현재 회원가입 API(POST /login/sign-up/kakao)는 필수 약관 3개가 모두 "Y"여야만 회원 생성 가능합니다.

필요한 변경:
- 약관 없이도 회원 생성 가능하도록 API 수정
- 약관 동의를 나중에 업데이트하는 API 추가

예상 작업량: 백엔드 30분~1시간, 프론트엔드 30분~1시간

8. 다음 단계: 비회원 견적 신청

지금까지의 개선은 "가입 과정을 쉽게 만드는 것"입니다.

하지만 진짜 해결책은 로그인 없이도 견적을 받을 수 있게 하는 것입니다.

비회원 견적 신청 도입 시
