PART 3. SEO 개선 전략 (Before → After)

3.1 핵심 전략: Server Component 분리

왜 Server Component가 필요한가?
Next.js 14에서 generateMetadata 함수는 Server Component에서만 작동합니다.
현재 상품 페이지는 'use client'로 되어 있어 SEO 메타데이터를 생성할 수 없습니다.

해결 방법: Server + Client Component 분리 패턴

Server Component (page.tsx)
  • SEO 메타데이터 생성
  • 초기 데이터 fetch (SSR)
  • Structured Data (JSON-LD)
  • 'use client' 없음
Client Component (ProductDetailClient.tsx)
  • useState, useEffect 사용
  • 사용자 인터랙션 처리
  • 실시간 UI 업데이트
  • 'use client' 필수
