3. 왜 이탈하나? - 기존 문제점

비회원이 상품 검색 | v 딜 리스트 확인 (가격, 통신사, 지역 모두 노출) | v --------- 정보를 다 봤음 -> "가입할 이유가 없네" 딜 클릭 -> 바로 Deal 페이지 이동 | v --------- 회원가입 유도 요소 없음 가입 없이 이탈
문제 1: 모든 정보가 공개되어 가입할 이유가 없음

기존에는 비회원에게 모든 딜 정보를 그대로 보여줬습니다.

  • 가격, 통신사, 지역 모두 노출
  • 정보를 다 확인했으니 가입할 필요성을 못 느낌
  • "가격만 보고 나가면 되지" 심리
문제 2: 회원가입 유도 요소가 없음

기존에는 회원가입을 유도하는 장치가 없었습니다.

  • 하단 고정 CTA 버튼 없음
  • 쿠폰 배너 없음
  • 시세 대비 절감액 표시 없음
  • 비회원도 회원과 동일하게 정보 확인 가능
문제 3: Welcome 페이지에 혜택이 안 보임

기존 Welcome 페이지:

  • "카카오로 로그인" 버튼만 덜렁 있음
  • 가입하면 뭐가 좋은지 설명이 없음
  • 쿠폰 혜택은 있는데 눈에 안 띔

4. 개선 내용: LowestDealSection

핵심 전략
1. 3위 가격을 공개해서 가치를 증명한다 (1~2위는 "가격 보러가기"로 유도)
2. 친근한 바텀시트로 가입 유도 (강제 이동 대신 부드러운 안내)
3. 5만원 쿠폰 혜택을 명확하게 보여준다
개선 1: 정보 차등 공개 + 가입 유도 장치

변경 전

모든 정보 그대로 노출
- 모든 딜 가격 공개
- 통신사, 지역 모두 공개
- 가입할 이유가 없음

변경 후

1~2위: 세부정보 노출 + 가격 숨김
- 통신사, 지역 정보 공개
- 가격 대신 "가격 보러가기" 표시
- 클릭 시 가입 안내 바텀시트

3위: 전체 노출
- 가격까지 모두 공개
- "이 정도면 진짜 싸네" 가치 증명

기존에는 모든 정보가 공개되어 가입할 이유가 없었습니다. 개선 후에는 3위 가격을 노출해서 가치를 증명하고, 1~2위 가격을 보려면 가입하도록 유도합니다.

개선 2: 가입 안내 바텀시트

변경 전

딜 클릭 시 바로 Welcome 페이지 이동

- 갑작스러운 페이지 전환
- 사용자 흐름 끊김

변경 후

가입 안내 바텀시트 표시

- "감사합니다" 친근한 타이틀
- 가격 정보 제공 이유 설명
- "무료로 맞춤 견적 받기" CTA

비회원이 딜을 클릭하거나 필터를 사용하려 할 때, 바로 페이지를 이동시키지 않고 친근한 톤의 바텀시트를 띄워서 가입을 유도합니다.

  • "저희가 수집한 가격 정보를 무료로 제공해 드리고 있어요"
  • "카카오톡으로 간편하게 가입해 주신 분들께만 보여드리고 있어요"
  • "가입 즉시 내 조건에 맞는 맞춤 견적을 확인하실 수 있어요"
개선 3: 필터 클릭 시 가입 유도

비회원이 필터(지역, 가입유형, 통신사, 요금제)를 클릭하면 가입 안내 바텀시트를 표시합니다.

  • 필터 기능은 회원 전용
  • 클릭 시 바로 이동하지 않고 안내 후 선택 유도
  • 사용자가 "더 세밀하게 검색하고 싶다" → 가입 동기
개선 4: 시세 대비 절감액 표시

상품 선택 시 타이틀에 시세 대비 절감액을 표시합니다.

  • "오늘 갤럭시 Z 폴드7, 시세보다 32만원 저렴해요"
  • 시세 = 출고가 - 공시지원금
  • 절감액 = 시세 - 최저가
  • 가치를 숫자로 명확하게 전달
개선 5: 신규가입 쿠폰 배너

비회원에게 쿠폰 혜택을 눈에 띄게 보여줍니다.

  • "신규가입 최대 5만원 추가할인"
  • "신규가입 . 첫구매 . 친구초대 혜택" 서브 문구
  • 상품 검색 전/후 모두 표시
  • 클릭 시 Welcome 페이지로 이동
개선 6: 지금 뜨는 상품

검색 전/후 화면에 인기 상품 목록을 표시해서 상품 탐색을 유도합니다.

  • 1~5위: viewCount(조회수) 상위 5개 상품
  • 6~10위: 나머지 상품 중 랜덤 5개
  • "클릭하면 바로 시세를 확인할 수 있어요" 서브 문구
  • 클릭 시 해당 상품 자동 선택 + 딜 로드
  • GA4 이벤트: popular_search_click (rank, product_name 포함)

쿠폰 혜택 명시

변경 전

10. 결론

핵심 전략

1. 가치 증명: 3위 가격 공개로 "이 서비스가 진짜 싸다"를 바로 보여줌
