3. 왜 이탈하나? - 기존 문제점

비회원이 상품 검색 | v 가격 확인하려고 딜 클릭 | v --------- 모든 정보 블러 처리 -> "뭐지?" 당황 딜 리스트 (모두 블러) | v --------- "본인인증" 문구 -> 부담감 Welcome 페이지 이동 | v --------- 혜택이 안 보임 -> "왜 가입해야 해?" 60% 이탈
문제 1: 모든 정보가 블러 처리됨

기존에는 비회원에게 모든 딜 정보를 블러 처리했습니다.

  • 1위 가격도 안 보임 -> "이 서비스가 싼지 비싼지" 판단 불가
  • 어떤 통신사, 어떤 지역인지 전혀 모름
  • 가치를 확인할 수 없으니 가입할 이유가 없음
문제 2: "본인인증" 문구의 부담감

기존 CTA 버튼 문구: "본인인증하고 맞춤견적 확인"

  • "본인인증" = 무거운 절차라는 인식
  • 실제로는 카카오 로그인 3초면 되는데, 문구 때문에 이탈
  • 사용자는 번거로운 절차를 피하고 싶어함
문제 3: Welcome 페이지에 혜택이 안 보임

기존 Welcome 페이지:

  • "카카오로 로그인" 버튼만 덜렁 있음
  • 가입하면 뭐가 좋은지 설명이 없음
  • 쿠폰 혜택은 있는데 눈에 안 띔

4. 개선 내용: LowestDealSection

핵심 전략
1. 1위 가격은 공개해서 가치를 증명한다
2. "본인인증" 대신 "3초 만에"로 부담감을 줄인다
3. 5만원 쿠폰 혜택을 명확하게 보여준다
개선 1: 1위 딜 가격 공개

변경 전

모든 딜 블러 처리
- 1위 가격도 안 보임
- 통신사, 지역 전부 블러
- "뭐가 좋은지 모르겠다"

변경 후

1위 딜 가격 공개
- 할부원금 그대로 노출
- "시세보다 XX만원 저렴" 표시
- 세부정보(통신사, 지역)만 블러

1위 가격을 보여주면 사용자가 "아 이 서비스 진짜 싸구나"를 바로 인지합니다. 세부 정보는 블러 처리해서 "더 알고 싶으면 가입하세요"를 유도합니다.

개선 2: CTA 버튼 문구 변경

변경 전

"본인인증하고 맞춤견적 확인"

- "본인인증" = 부담스러운 절차
- 길고 복잡한 느낌

변경 후

"3초 만에 맞춤견적 받기"

- "3초" = 빠르고 간편함
- 실제 카카오 로그인 소요 시간

개선 3: 하단 고정 CTA

비회원이 상품을 검색하고 딜 목록을 볼 때, 화면 하단에 고정된 CTA 버튼을 표시합니다.

  • 스크롤해도 항상 보이는 위치
  • "지원금 변동으로 가격이 수시로 바뀌어요" 안내 문구
  • "3초 만에 맞춤견적 받기" 버튼
  • 버튼 클릭 시 Welcome 페이지로 이동
개선 4: 시세 대비 절감액 표시

상품 선택 시 타이틀에 시세 대비 절감액을 표시합니다.

  • "오늘 갤럭시 Z 폴드7, 시세보다 32만원 저렴해요"
  • 시세 = 출고가 - 공시지원금
  • 절감액 = 시세 - 최저가
  • 가치를 숫자로 명확하게 전달
개선 5: 신규가입 쿠폰 배너

비회원에게 쿠폰 혜택을 눈에 띄게 보여줍니다.

  • "신규가입 최대 5만원 추가할인"
  • "신규가입 . 첫구매 . 친구초대 혜택" 서브 문구
  • 상품 검색 전/후 모두 표시
  • 클릭 시 Welcome 페이지로 이동

10. 결론

핵심 전략

1. 가치 증명: 1위 가격 공개로 "이 서비스가 진짜 싸다"를 바로 보여줌
2. 부담감 제거: "본인인증" -> "3초 만에"로 심리적 장벽 낮춤
3. 혜택 명시: 5만원 쿠폰을 눈에 띄게 표시해서 가입 동기 부여
4. 노출 증가: 하단 고정 CTA로 언제든 가입 가능하게
