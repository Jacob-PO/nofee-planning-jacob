요금제 그룹핑 패치

가격대별 요금제 그룹핑을 통한 상품 필터링 및 가격 테이블 노출 개선

문서 유형
패치 기획안
우선순위
중간 (Medium)
목표 배포일
2025.11.25 ~ 11.29

개요

현재 동네성지 서비스의 요금제 필터는 각 통신사의 정확한 요금제 금액을 기준으로 필터링되어, 고객이 원하는 가격대의 상품을 한눈에 비교하기 어렵습니다. 본 패치를 통해 요금제를 가격대별로 그룹핑하여, 고객이 가격대를 선택하면 해당하는 모든 통신사의 요금제 상품을 통합하여 노출합니다.

핵심 변경사항
• 요금제 필터를 정확한 금액에서 가격대 그룹(예: 11만원대, 10만원대)으로 변경
• 상품 가격 테이블을 요금제 그룹 단위로 통합 노출하여 통신사별 비교 용이
• 각 통신사별 요금제 금액이 다르더라도 동일 가격대 그룹으로 묶어 표시

배경 및 목적

현재 문제점

  • 통신사별 요금제 금액 상이: 동일 가격대임에도 SKT는 109,000원, KT는 110,000원, LG는 115,000원으로 각기 다른 금액 체계
  • 필터링 불편: 고객이 "11만원대" 요금제를 원할 때 각 통신사별로 별도 검색 필요
  • 비교 어려움: 가격 테이블에서 통신사별 동일 가격대 상품을 한눈에 비교하기 어려움
  • 고객 혼란: "왜 109,000원과 110,000원이 따로 있지?" 같은 의문 발생

개선 목표

  • 직관적인 필터링: "11만원대", "10만원대" 등 가격대 단위로 요금제 선택
  • 통합 비교: 하나의 가격대 그룹 선택으로 모든 통신사의 해당 가격대 요금제 비교
  • UX 개선: 고객이 원하는 예산 범위 내에서 최적의 상품 빠르게 탐색
  • 검색 효율성: 필터 선택 횟수 감소로 상품 검색 시간 단축

그룹핑 규칙

• 각 가격대 그룹은 통신사별 유사한 요금제를 하나로 묶습니다
• 일부 통신사에서 해당 가격대 요금제가 없는 경우 "-"로 표시
• 그룹 이름은 대표 가격대를 사용 (예: 11만원대, 10만원대)
• 고객이 "11만원대"를 선택하면 → SKT 109,000원, KT 110,000원, LG 115,000원 요금제를 모두 조회

2. 상품 가격 테이블 노출 변경

변경 전: 요금제 금액별 노출 (AS-IS)

• 109,000원 선택 시 → SKT 상품만 표시
• 110,000원 선택 시 → KT 상품만 표시
• 115,000원 선택 시 → LG 상품만 표시
• 통신사별 개별 검색 필요 (총 3번 선택)

변경 후: 요금제 그룹별 노출 (TO-BE)

• 11만원대 선택 시 → SKT(109,000원), KT(110,000원), LG(115,000원) 모두 표시
• 가격 테이블에서 통신사별 가격 한눈에 비교
• 한 번의 선택으로 모든 옵션 확인 (1번만 선택)
• 각 셀에 실제 요금제 금액도 함께 표기하여 투명성 확보

3. 가격 테이블 UI 변경

  • 그룹 헤더 추가: 선택한 요금제 그룹명을 가격 테이블 상단에 표시 (예: "가격 (11만원대 요금제)")
  • 통신사별 실제 금액 표기: 각 통신사 셀에 실제 요금제 금액 병기 (예: 작은 글씨로 "109,000원")
  • 비교 용이성: 동일 행에 모든 통신사 가격을 배치하여 직관적 비교 가능
  • 일관성 유지: 기존 가격 테이블의 레이아웃과 스타일은 최대한 유지

사용자 시나리오

시나리오 1: 가격대 기반 상품 검색

  1. 고객이 상품 상세 페이지에서 "서울 강남구" 지역 선택
  2. "256GB" 용량 선택
  3. 요금제 필터에서 "11만원대" 선택
  4. 가격 테이블에 SKT(109,000원), KT(110,000원), LG(115,000원)의 모든 요금제 상품이 한눈에 표시됨
  5. 통신사별 공시지원금, 선택약정 가격을 비교하여 최적 상품 선택
  6. 원하는 가격 셀 클릭 → 매장 검색 결과 표시
Before: 3번의 선택 필요
109,000원 선택 → SKT 확인 → 110,000원 선택 → KT 확인 → 115,000원 선택 → LG 확인

After: 1번의 선택으로 완료
11만원대 선택 → SKT, KT, LG 모두 한눈에 확인

시나리오 2: 가격대 변경을 통한 비교

  1. 고객이 "11만원대" 요금제 검색 후 가격 확인
  2. 예산이 부족하다고 판단하여 요금제 필터를 "8만원대"로 변경
  3. 가격 테이블이 즉시 업데이트되어 KT(90,000원), SKT(89,000원), LG(95,000원) 요금제 표시
  4. 더 저렴한 가격대에서 통신사별 옵션 비교 가능

시나리오 3: 일부 통신사 미지원 케이스

  1. 고객이 "5만원대" 요금제 선택
  2. 가격 테이블에 SKT만 표시되고, KT와 LG는 "-" 또는 "해당 없음"으로 표시
  3. 고객은 "5만원대는 SKT만 있구나"라고 직관적으로 이해

시세표 이미지 다운로드 기능 연동

시세표 이미지 다운로드 기능은 최고가 요금제 그룹(11만원대)을 기준으로 샘플 이미지가 생성됩니다. 이는 매장에서 가장 많이 판매되는 프리미엄 요금제 중심으로 시세표를 공유할 수 있도록 하기 위함입니다.

시세표 이미지 생성 기준

• 기본 요금제: 11만원대 그룹 (SKT 109,000원, KT 110,000원, LG U+ 115,000원)
• 샘플에 포함되는 상품: 매장에서 설정한 주요 판매 상품 (갤럭시 S24, 아이폰 15 등)
• 가격 정보: 11만원대 요금제 기준 통신사별 공시지원금, 선택약정 가격 표시
• 이미지 형식: 1080px 너비, SNS 공유 최적화 디자인
