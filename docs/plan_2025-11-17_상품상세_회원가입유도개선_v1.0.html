검색 영역 마스킹 확대 + 하단 고정 버튼 추가로 회원가입 전환율 향상

패치 기획안

높음 (High)

1. 현재 문제 상황

문제점

상품 상세페이지에서 최저가 지역을 필터에 미리 노출하면, 해당 지역을 희망하지 않는 방문자가 회원가입하지 않고 즉시 이탈하는 현상이 발생하고 있습니다.

구체적인 문제:

  • 필터 영역에서 특정 지역의 최저가만 보임 → 다른 지역 희망 유저 이탈
  • 회원가입 유인이 약함 → 전환율 저조
  • 마스킹 영역이 가격 필터만 가려져 있어 부분적으로 정보 노출됨

2. 해결 방안

핵심 변경사항
  • "동네성지 최저가 검색" 전체 영역 마스킹 처리 (비회원만 해당)
  • 지역, 용량/요금제, 가격 필터 모두 로그인 후에만 확인 가능
  • 하단 고정 버튼 추가 (비회원만 노출, 로그인 완료 시 자동 제거)

기대 효과:

  • 검색 영역 마스킹: 모든 검색 정보는 로그인 후에만 확인 가능
  • 하단 고정 버튼: 페이지 어디서든 회원가입 유도 (접근성 향상)
  • 이중 전환 포인트: 마스킹 영역 + 하단 버튼으로 회원가입 기회 극대화

5. 기대 효과

정량적 효과
  • 회원가입 전환율 15~20% 향상: 이중 전환 포인트로 접촉 기회 증가
  • 상세페이지 이탈률 5~10% 감소: 하단 고정 버튼으로 접근성 개선
  • 견적 신청 전환율 향상: 언제든 버튼 클릭 가능 (스크롤 무관)
정성적 효과
  • 명확한 정보 구분: 회원/비회원 간 정보 접근 권한 명확화
  • 사용자 경험 개선: 페이지 어디서든 회원가입 진입 가능
  • 브랜드 가치 향상: 회원 전용 혜택의 가치 강화
  • 전환 기회 극대화: 마스킹 영역 + 하단 버튼으로 이중 유도
